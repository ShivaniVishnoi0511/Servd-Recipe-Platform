# 🍲 Full Stack AI Recipe Platform

A modern full-stack AI-powered recipe generator built using **Next.js**, **Neon PostgreSQL**, **Strapi**, and **Google Gemini API**.  
The platform allows users to generate intelligent recipes based on ingredients using AI.

---

## 🚀 Features

- 🧠 AI-powered recipe generation using Gemini API  
- 🔍 Smart ingredient-based search  
- ⚡ Fast and optimized frontend with Next.js  
- 🎨 Beautiful UI using Tailwind CSS + Shadcn UI  
- 🛡️ Backend CMS using Strapi  
- 🗄️ PostgreSQL database powered by Neon DB  
- 🔐 Security protection using Arcjet  
- 📱 Fully responsive design

---

## 🛠 Tech Stack

### Frontend
- Next.js
- Tailwind CSS
- Shadcn UI

### Backend
- Strapi CMS
- Node.js

### Database
- Neon DB (PostgreSQL)

### AI & Security
- Google Gemini API
- Arcjet Security
